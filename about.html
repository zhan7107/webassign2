About

This is a Air Quality Web App that is interactive, responsive, and allow a user to investigate air quality from various geographical regions.

This project will incorporate Vue JS, as our javascript framework.

We use Google Map API for our maps and Open AQ Platform for retrieve data for air quality measures.

API used are vue2-datepicker for getting the calendar working and axios for getting our promise from Open AQ
